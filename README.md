# うんち💩（フリマ損益計算サイト）

## iOSアイコン設定について

iOSのホーム画面にショートカットを追加する際のアイコンは、以下の場所に配置してください：

```
/static/apple-touch-icon.png
```

推奨仕様：
- サイズ：180x180px または 512x512px
- 形式：PNG（透明背景可）
- ファイル名：apple-touch-icon.png

アイコンファイルを配置後、アプリケーションを再起動すると反映されます。

## 主な修正点

### 1. 入力フォームの改善
- 各入力欄に明確なplaceholderを設定
- 必須項目を明示
- 日付入力にplaceholderを追加

### 2. レスポンシブデザイン
- スマホ表示での横スクロール対応
- テーブルの最適化
- タッチ操作の改善

### 3. データ表示の改善
- 横方向への展開
- 手数料列の追加
- 利益の色分け表示（黒字：緑、赤字：赤）

### 4. 商品名の長さ対応
- 長い商品名は「...」で省略
- タップで全文表示
- ホバー効果

### 5. タグの色分け
- 購入先プラットフォームごとに色を設定
- 商品分類ごとに色を設定
- 販売サイトも色分け

### 6. 手数料計算の修正
- ラクマ：10%
- ヤフーフリマ：5%
- メルカリ：10%

### 7. 利益計算の明確化
- 利益 = 販売価格 - 仕入価格 - 送料 - 手数料
- 利益率 = 利益 ÷ 仕入価格 × 100
- 総利益の表示を明確化

## 起動方法

```bash
python app.py
```

ブラウザで `http://localhost:5000` にアクセスしてください。
